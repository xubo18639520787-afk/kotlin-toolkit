WaysOfReading

data class WaysOfReading(val visualAdjustments: AccessibilityMetadataDisplayGuide.WaysOfReading.VisualAdjustments = VisualAdjustments.UNKNOWN, val nonvisualReading: AccessibilityMetadataDisplayGuide.WaysOfReading.NonvisualReading = NonvisualReading.NO_METADATA, val nonvisualReadingAltText: Boolean = false, val prerecordedAudio: AccessibilityMetadataDisplayGuide.WaysOfReading.PrerecordedAudio = PrerecordedAudio.NO_METADATA) : AccessibilityMetadataDisplayGuide.Field

The ways of reading display field is a banner heading that groups together the following information about how the content facilitates access.

https://w3c.github.io/publ-a11y/a11y-meta-display-guide/2.0/guidelines/#ways-of-reading

Parameters

visualAdjustments

Indicates if users can modify the appearance of the text and the page layout according to the possibilities offered by the reading system.

nonvisualReadingAltText

Indicates whether text alternatives are provided for visuals.

prerecordedAudio

Indicates the presence of prerecorded audio and specifies if this audio is standalone (an audiobook), is an alternative to the text (synchronized text with audio playback), or is complementary audio (portions of audio, (e.g., reading of a poem).

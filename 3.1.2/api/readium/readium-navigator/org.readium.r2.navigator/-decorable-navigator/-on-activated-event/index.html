OnActivatedEvent

data class OnActivatedEvent(val decoration: Decoration, val group: String, val rect: RectF? = null, val point: PointF? = null)

Holds the metadata about a decoration activation interaction.

Parameters

decoration

Activated decoration.

group

Name of the group the decoration belongs to.

rect

Frame of the bounding rect for the decoration, in the coordinate of the navigator view. This is only useful in the context of a VisualNavigator.

point

Event point of the interaction, in the coordinate of the navigator view. This is only useful in the context of a VisualNavigator.
